## Task 2: Token Quota Pre-Check Fix

- [ ] **Step 1: Add token estimation helper**

Add a private helper near `send_message`:
```rust
fn estimate_input_tokens(content: &[ai_provider::Content]) -> u64 {
    let chars: usize = content.iter().map(|c| match c {
        ai_provider::Content::Text(t) => t.len(),
        ai_provider::Content::Image { .. } => 1024, // rough image token estimate
        ai_provider::Content::Audio { .. } => 1024,
        ai_provider::Content::Video { .. } => 2048,
        _ => 0,
    }).sum();
    (chars / 4).max(1) as u64
}
```
